# Academic Timetable Conflict Detection System

## 📌 Description
This project is a Python-based application to manage academic timetables and detect scheduling conflicts.

## ⚙️ Technologies Used
- Python
- Tkinter (GUI)
- SQLite (Database)

## 🚀 Features
- Add timetable entries
- Detect conflicts (Faculty, Room, Class)
- View timetable
- Update entries
- Delete entries

## 🗄️ Database
The project uses SQLite with a single table:
- timetable (class, subject, faculty, room, day, time)

## ▶️ How to Run
1. Install Python
2. Run:
   ```bash
   python timetable.py
